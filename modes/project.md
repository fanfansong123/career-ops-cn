## "面试包"要求

对每个通过的项目：
1. **一页纸**: 产品 + 架构 + 指标 + 评估方案
2. **演示**: 在线 URL 或 2分钟录屏
3. **复盘**: 什么做对了、什么做错了、如何改进

## 80/20 计划

- 第1周 → MVP + 核心指标
- 第2周 → 打磨 + 面试包

## 结论

- **值得做** → 含每周里程碑的计划
- **跳过** → 原因及替代建议
- **转向 [替代方案]** → 更有影响力的变体
